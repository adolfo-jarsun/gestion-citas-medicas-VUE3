# 🏥 Sistema de Gestión de Citas Médicas

Sistema web desarrollado con **Vue 3** para la gestión de pacientes, médicos y citas médicas. Implementa autenticación de usuarios, CRUD completo, relaciones entre entidades, y protección de rutas.

## 📋 Requisitos del Sistema

- **Node.js**: v20.19.0 o superior
- **npm**: v10.x o superior
- **Git**: Para clonar el repositorio

> ⚠️ **Importante**: Este proyecto ha sido probado y es compatible con Node.js v20.19.0. El uso de otras versiones puede causar problemas de compatibilidad.

## 🚀 Tecnologías Utilizadas

| Tecnología | Versión | Propósito |
|------------|---------|------------|
| Vue 3 | ^3.5.32 | Framework frontend |
| Vite | ^8.0.8 | Bundler y servidor de desarrollo |
| Vue Router | ^4.x | Navegación SPA y rutas dinámicas |
| Pinia | ^3.0.4 | Estado global y autenticación |
| Axios | ^1.15.1 | Cliente HTTP para API |
| JSON Server Auth | ^2.1.0 | Backend fake con autenticación JWT |
